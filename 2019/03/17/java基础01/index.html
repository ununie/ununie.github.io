java基础01

发表于 2019-03-17

1、jvm、jre、jdk

jvm：java虚拟机

jre：java运行环境，包含了java虚拟机和java程序所需要的核心类库。如果运行一个java程序，只需要jre即可

jdk：java开发工具包，包含了jre、编译工具javac.exe、打包工具jar.exe

2、jdk下载安装

配置环境变量，JAVA_HOME

3、java语法

1、关键字

  • goto和const作为保留字
  • 关键字都是小写

2、标识符

  • 给类、接口、方法、变量等命名时使用的字符序列。

  • 不能以数字开头

  • 不能是关键字

3、注释

  • 单行注释//注释内容
  • 多行注释/ 注释内容 /
  • 文档注释/ 注释内容 /

作用：提高阅读性，注释掉内容帮助调试

4、常量

  • 字面值常量：字符串、整数、小数、字符常量、布尔常量、null
  • 自定义常量

1、进制

  • 整数常量的四种表现形式：
    • 二进制：以0b开头
    • 八进制：以0开头
    • 十进制：默认
    • 十六进制：以0x开头，0-9以及a-f（大小写均可）
    • 进制之间转换：十进制到其它n进制，用数字除以n，余数倒序

一个字节（byte）等于8位（bit）

1K=1024byte

1M=1024K

2、原码、反码、补码

所有数据运算都是用补码进行的

原码：就是二进制表示法，最左边为0表示整数，1表示负数正数

反码：正数反码与原码相同；负数反码是原码逐位取反符号位除外

补码：正数补码与原码相同；负数补码是反码末位加1

5、变量

数据类型 变量名=变量值

6、数据类型

基本数据类型：整型、浮点型、字符型、布尔型

引用数据类型：类、接口、数组

1、基本类型

整型 占用字节个数 范围
byte 1 -128-127
short 2 -2^15-2^15-1
int 4 -2^31-2^31-1
long 8 -2^53-2^53-1
浮点型
float 4 -3.403e38-3.403e38
double默认 8 -1.798e308-1.798e308
字符型
char 2
布尔型
boolean 1 默认false

长整型long后缀用L标记

单精度浮点型float后缀用F标记

2、使用变量注意事项

变量的作用域，变量所定义在的大括号内

相同作用域不能定义同名变量

没有初始化的变量不能直接使用

默认转换：

byte、short、char——int——long——float——double

byte、shrot、char相互之间不转换，他们参与运算都转为int进行运算。即便是两个byte之间进行运算也是先转int

强制转换

long转到float为什么可以？

整型和浮点型底层存储结构不同；float表示的数据范围比long的范围大

java中的char类型可以存一个中文汉字吗？

可以，java中的char类型占用2个字节。汉字一般占两个字节。java采用的是unicode编码

7、运算符

算术运算符、赋值运算符、比较运算符、逻辑运算符、位运算符、三目运算符

位运算符

运算符 示例
<< 左移 1001<< 10010
>> 右移 1001>> 1100
>>> 无符号右移 1001>> 0100
& 与运算 有0则0
\ 或运算 有1则1
^ 异或 相同则0，不同则1
~ 取反 按位取反

a^b^b结果为a，可以用作简单的数据加密

m<<n = m*2^n

m>>n = m/2^n

加法操作a+b相当于a^b + (a&b)<<1 递归调用直到a&b=0

顺序语句、条件语句、循环语句

switch语句的取值byte、int、short、char。JDK5以后可以是枚举，JDK7以后可以是字符串
